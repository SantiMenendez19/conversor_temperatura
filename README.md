# Conversor de Temperatura
Interfaz para calcular la temperatura en grados Celsius, Kelvin y Fahrenheit realizado en Python con GUI Tkinter

# Modo de uso

Su uso es simple , introduce un valor en cualquiera de las entradas (solo numeros y decimales con punto "." ) y pulsa "Convertir" para conocer su valor en grados en otras mediciones de temperatura. Si quieres conocer otro valor usando otra medicion usa "Limpiar" para borrar las entradas y escribe el valor en la entrada que vayas a usar.

Es una version simple para ir practicando primeros proyectos simples en Python
